# Projeto de Análise de Vendas

Projeto de análise de dados utilizando SQL e Power BI para geração de insights de vendas e construção de dashboard interativo.

## Ferramentas
- PostgreSQL
- Power BI

### Análises realizadas
- Faturamento total
- Vendas por produto
- Vendas por cidade
- Evolução das vendas
